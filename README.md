# 🎰 파이썬 로또 번호 생성기 (Python Lotto Generator)

Flask를 기반으로 제작된 간편한 로또 번호 추첨 웹 애플리케이션입니다.

## 🚀 기능 (Features)

- **랜덤 번호 추첨**: 1부터 45까지의 숫자 중 중복 없이 6개의 번호를 무작위로 추출 및 정렬합니다.
- **다중 게임 생성**: 사용자가 입력한 원하는 게임 수만큼 한 번에 번호 세트를 생성하여 5게임 단위로 묶어서 보여줍니다.
- **당첨 판별 로직**: 당첨 번호 및 보너스 번호와 비교하여 1등부터 5등, 꽝을 정확하게 계산하는 코어 기능이 포함되어 있습니다.
- **직관적인 웹 UI**: Flask를 통한 서버 구동 후, 브라우저에서 편리하게 수량을 입력하고 결과를 얻을 수 있습니다.
- **다양한 디바이스 지원**: 반응형 웹페이지 구현 등 웹과 모바일 환경 모두에서 원활하게 구동 가능합니다.

## 🛠️ 기술 스택 (Tech Stack)

- **Backend**: Python 3.10.14, Flask
- **Frontend**: HTML

## 📁 프로젝트 폴더 구조 (Project Structure)

```text
.
├── app.py             # Flask 웹 서버 구동 및 페이지 라우팅 처리
├── lotto.py           # 로또 번호 생성 및 등수 판별 시스템 핵심 로직
├── test_logic.py      # 당첨 판별 로직이 올바른지 검증하는 테스트 파일
└── templates/         # HTML 뷰(View) 파일들
    ├── index.html     # 메인 페이지 (구입할 게임 수 입력창)
    └── result.html    # 추첨 결과 페이지 (생성된 번호 출력)
```

## ⚙️ 실행 방법 (Getting Started)

1. **필요 패키지 설치** (가상환경 사용을 권장합니다)

   ```bash
   pip install flask
   ```

2. **서버 실행하기**

   ```bash
   python app.py
   ```

3. **브라우저 접속**  
   서버가 정상적으로 켜지면 웹 브라우저 주소창에 `http://localhost:5000` 또는 `http://127.0.0.1:5000`을 입력하여 접속합니다.

## ✅ 테스트 (Testing)

로또 등수 판별 로직이 규정에 맞게 작동하는지 확인하기 위해 포함된 테스트 파일을 실행할 수 있습니다.

```bash
python test_logic.py
```

> 결과로 `All tests passed!` 문구가 나타나면 로직에 이상이 없는 것입니다.
